# KeyScan v1.0.0 发布说明

这是 KeyScan 的首个公开发布版本。

## 版本信息

- 应用名称：KeyScan
- 版本号：1.0.0
- versionCode：1
- 包名：com.secureqr.scanner
- 系统要求：Android 8.0 及以上

## 主要功能

- 支持二维码和条形码实时扫描，可保存扫码历史。
- 支持从相册导入二维码图片并自动识别。
- 支持生成二维码，可自定义颜色、圆点、圆角和 Logo。
- 内置密码账本，支持保存账号密码、搜索、强密码生成和扫码填充。
- 内置 OTP 双因子认证器，支持扫码添加、手动添加和验证码一键复制。
- 内置随机密码生成器，支持自定义长度和字符类型。
- 支持扫码历史、生成历史、密码账本和 OTP 数据管理。
- 支持 WebDAV 加密同步，包含主备双备份通道，方便换机恢复。
- 支持数据保险功能，可导出加密备份、恢复数据并验证完整性。
- 支持亮色、深色和跟随系统主题。
- 内置隐藏小游戏：推推拼图、推箱子、俄罗斯方块。

## 安全说明

- 密码账本、OTP 密钥等敏感数据会在本地加密存储。
- WebDAV 备份会先在本地加密，再上传到用户自己配置的云端空间。
- KeyScan 不内置广告 SDK，不主动上传用户数据到开发者服务器。
- 请妥善保存自己的 WebDAV 账号、备份密码和恢复密钥。
